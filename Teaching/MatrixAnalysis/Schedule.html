Homework Hints

  • 4.1.1: For this and other problems in which you have to identify whether a statement is true or not, either prove the statement is true or provide a simple counterexample with accompanying explanation.
  • 4.1.11: To show two sets are equal show every element of one set is contained in the other, and vice-versa. Convert statements about sets into equations inviolving vectors in those sets.
  • 3.2.6b: This is analogous to the proof that any function can be uniquely expressed as the sum of an odd and an even function. Hint for uniqueness: Show that if a square matrix is both symmetric and skew-symmetric then it is zero. You can do this just using matrix algebra, that is working with the matrix as an entity on its own right. It is not necessary to manipulate the entries of the matrix.
  • 4.1.6de: See Example 4.1.7 Meyer pages 165-166.
  • 3.5.3: Use the grey box on Meyer page 98.
  • 3.6.5: See hint to 4.1.1
  • 3.9.6: See hint to 4.1.1
  • 4.2.2b: Here is a rephrasing of the question in case you need it. Suppose that Ax=b is consistent. Prove that Ax=b has a unique solution if and only if N(A) = {0}.
  • 4.2.7: Show that if Ax=0 then the length of x is zero.
  • 4.3.6: First understand why result is true in 3x3 case. Then prove in general.
  • 4.3.10: Find a vector in the nullspace of A
